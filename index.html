1. What is HTML?

HTML stands for HyperText Markup Language that is used to create and preview documents in web browser.

2. What is a basic structure of an HTML document?

HTML consists of a series of elements(tags) which each of them has it own way of showing and using the content. It starts with "!DOCTYPE html" tag which indicates the version of HTML. The "html" tag which is the root of the webpage. The "head" tag which indicates the meta info of the webpage which users can not see it. The "body" tag where is the main content is wirtten.
